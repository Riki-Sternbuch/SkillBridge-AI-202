## Project Overview
SkillBridge AI is a specialized web application designed to help job seekers align their resumes with specific job descriptions. Using AI-driven analysis, the platform identifies professional gaps, suggests resume optimizations, and provides tailored interview preparation based on the requirements of the Israeli tech market.

## Key Features
*   **Resume-Job Matching:** Automated comparison between uploaded resumes (PDF/DOCX) and job descriptions.
*   **Gap Analysis:** Identification of missing technical and soft skills with direct links to relevant learning resources.
*   **Resume Optimization:** AI-generated bullet points for improving resume impact.
*   **Interview Preparation:** Customized interview questions based on identified skill gaps.
*   **Persistence:** Local storage integration to save and review previous analyses.
*   **Full RTL Support:** Modern Hebrew user interface designed for the local market.

## Technical Implementation
The project was developed using the **Base44 AI platform**, focusing on:
*   **AI Orchestration:** Designing complex logic for Hebrew natural language processing (NLP).
*   **Prompt Engineering:** Developing structured instructions for the LLM to ensure accurate scoring and professional career insights.
*   **Frontend Design:** Implementation of a modern "Midnight Slate" UI with Glassmorphism effects and responsive layout.
*   **Data Extraction:** Client-side text processing from multiple file formats.

## Usage Instructions
1. Paste the target **Job Description** in the designated field.
2. Upload your **Resume** (PDF/DOCX).
3. Select the relevant **Industry** and **Experience Level**.
4. Click **"Analyze"** to generate a comprehensive compatibility report.
